إعداد حفظ البيانات في Google Sheets

مهم: لا يمكن لأي برنامج خارجي إنشاء Google Sheet على حسابك أو إرسال رابط جاهز للحفظ. يجب أن تنشئي أنت النموذج أو السكربت من حساب Google (مرة واحدة فقط).
جدولك الحالي:
فتح Google Sheet — لعبه 1

✅ الطريقة الأسهل — Google Form (دقيقتان)

  1. افتحي forms.google.com → نموذج فارغ
  2. أضيفي 6 أسئلة (إجابة قصيرة):
    الاسم — الجوال — المستوى — الموضوع — النتيجة — التاريخ
  3. Responses → Link to Sheets → اختاري جدول لعبه 1
  4. على كل سؤال: ⋮ → Get pre-filled link → اكتبي أي نص → انسخي الرابط
  5. من الرابط انسخي قيم entry.XXXXX لكل سؤال
  6. من زر Send → انسخي رابط النموذج وحوليه إلى:
    https://docs.google.com/forms/d/e/FORM_ID/formResponse
  7. أرسلي لنا: رابط formResponse + أرقام entry — نضيفها في .env

⚙️ الطريقة المتقدمة — Apps Script

  1. افتحي الجدول → Extensions → Apps Script
  2. الصقي كود google-apps-script/Code.gs
  3. Run → setupSheet ثم testAppend
  4. Deploy → Web app → Me + Anyone
  5. انسخي رابط /exec وأرسليه لنا

📩 بعد الإعداد

أرسلي أحد الخيارين:
  • رابط Google Form (formResponse) + أرقام entry
  • أو رابط Apps Script (/exec)
وسنضيفها في ملف .env ويعمل الحفظ تلقائياً.
